## 1. FUNÇÕES (~2 questões)

### Função Afim (1º grau) - f(x) = ax + b
- **a** = coeficiente angular (taxa de variação)
- **b** = coeficiente linear (corta eixo y)
- Raiz: x = -b/a

**Macete:** a > 0 = crescente. a < 0 = decrescente.

### Função Quadrática (2º grau) - f(x) = ax² + bx + c
- Vértice: x_v = -b/(2a), y_v = -Δ/(4a)
- Δ = b² - 4ac

**Macete:** a > 0 = concavidade p/ cima (mínimo). a < 0 = p/ baixo (máximo).

## 3. ANÁLISE COMBINATÓRIA (~1 questão)

| Conceito | Fórmula | Quando |
|----------|---------|--------|
| PFC | Multiplica possibilidades | Etapas sucessivas |
| Permutação | n! | Ordenar todos |
| Arranjo | n!/(n-p)! | Ordem importa |
| Combinação | n!/p!(n-p)! | Ordem NÃO importa |

**Macete:** "Ordem importa?" Se sim → Arranjo. Se não → Combinação.
