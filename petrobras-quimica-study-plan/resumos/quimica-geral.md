## 1. FUNÇÕES INORGÂNICAS (~7 questões)

### Ácidos
- Liberam H⁺ em água. pH < 7.
- Fortes: HCl, H₂SO₄, HNO₃. Fracos: H₂CO₃, H₃PO₄.
- **Sem oxigênio:** hidro + nome + ídrico (HCl = ácido clorídrico)
- **Com oxigênio:** -ico (mais O), -oso (menos O)

### Bases
- Liberam OH⁻ em água. pH > 7.
- Fortes: NaOH, KOH, Ca(OH)₂. Fraca: NH₄OH.
- Nome: Hidróxido de + metal

### Sais
- Ácido + Base → Sal + Água (neutralização)
- Nome: ânion + "de" + cátion. NaCl = cloreto de sódio

### Óxidos
- **Ácido:** não-metal + O (CO₂, SO₂) - reage com base
- **Básico:** metal + O (Na₂O, CaO) - reage com ácido
- **Anfótero:** Al₂O₃, ZnO - reage com ambos

### Fórmulas essenciais

| Conceito | Fórmula |
|----------|---------|
| Concentração comum | C = m/V (g/L) |
| **Molaridade** | **M = n/V = m/(MM×V)** |
| Título | τ = m₁/m |
| Diluição | **M₁V₁ = M₂V₂** |
| Mistura (mesmo soluto) | M_f = (M₁V₁ + M₂V₂)/(V₁+V₂) |
| Kps | Para A₂B₃: Kps = [A]²[B]³ |

### Propriedades Coligativas
- Dependem do NÚMERO de partículas, não da natureza.
- Quanto mais partículas, maior o efeito.

## 3. REAÇÕES E ESTEQUIOMETRIA (~6 questões)

### NOX - Regras
1. Substância simples = 0
2. H = +1 (exceto hidretos: -1)
3. O = -2 (exceto peróxidos: -1)
4. Soma NOX em composto neutro = 0
5. Soma NOX em íon = carga do íon

**Redox:** Quem perde e⁻ = oxida (NOX ↑) = agente redutor.  
Quem ganha e⁻ = reduz (NOX ↓) = agente oxidante.

### Estequiometria
1. Equação balanceada
2. Regra de três com coeficientes
3. Rendimento: multiplica resultado por %
4. Pureza: só % da massa é reagente real

## 4. MODELOS ATÔMICOS E TABELA (~1-2 questões)

### Evolução: Dalton → Thomson → Rutherford → Bohr → Schrödinger

### Números quânticos
- n = nível (1,2,3...)
- l = subnível (0=s, 1=p, 2=d, 3=f)
- m = orbital (-l a +l)
- s = spin (±½)

### Tabela Periódica
- **Eletronegatividade** cresce → direita e ↑ cima (F o maior)
- **Raio atômico** cresce ← esquerda e ↓ baixo

### Kps
Kps = s² (para AB) ou 4s³ (para AB₂) ou 27s⁴ (para A₂B₃)
