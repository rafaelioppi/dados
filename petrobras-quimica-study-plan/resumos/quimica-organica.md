## 1. HIDROCARBONETOS E PETRÓLEO

### Classificação das Cadeias
- **Aberta** (acíclica) x **Fechada** (cíclica)
- **Normal** x **Ramificada**
- **Saturada** (só simples) x **Insaturada** (dupla/tripla)

### Hidrocarbonetos

| Família | Fórmula | Exemplo |
|---------|---------|---------|
| Alcano | CₙH₂ₙ₊₂ | CH₃-CH₃ |
| Alceno | CₙH₂ₙ | CH₂=CH₂ |
| Alcino | CₙH₂ₙ₋₂ | CH≡CH |
| Aromático | C₆H₆ | Benzeno |

### Petróleo - Destilação Fracionada
**Frações (do + leve ao + pesado):**
Gás → GLP (C₃-C₄) → **Gasolina (C₅-C₁₀)** → Querosene → Diesel → Lubrificante → Asfalto

**Macete:** Quanto MAIOR a cadeia, MAIOR o ponto de ebulição.  
Quanto MAIS ramificada, MENOR o ponto de ebulição.

## 2. NOMENCLATURA IUPAC

**Prefixo** (quantos C) + **Infixo** (tipo ligação) + **Sufixo** (função)

| C | Prefixo |
|---|---------|
| 1 | met- |
| 2 | et- |
| 3 | prop- |
| 4 | but- |
| 5 | pent- |

| Ligação | Infixo |
|---------|--------|
| Simples | -an- |
| Dupla | -en- |
| Tripla | -in- |

**Regra:** Numerar a cadeia para dar menor nº à insaturação/ramificação.

## 3. FUNÇÕES ORGÂNICAS

| Função | Grupo | Exemplo |
|--------|-------|---------|
| **Álcool** | -OH | CH₃-OH (metanol) |
| **Fenol** | -OH no anel | C₆H₅-OH |
| **Éter** | -O- | CH₃-O-CH₃ |
| **Aldeído** | -CHO | CH₃-CHO |
| **Cetona** | -CO- | CH₃-CO-CH₃ |
| **Ácido** | -COOH | CH₃-COOH |
| **Éster** | -COO- | CH₃-COO-CH₃ |
| **Amina** | -NH₂ | CH₃-NH₂ |
| **Amida** | -CONH₂ | CH₃-CONH₂ |

### Reações Importantes
- **Esterificação:** Ácido + Álcool → Éster + Água
- **Oxidação:** Álcool primário → Aldeído → Ácido
