## 1. EQUILÍBRIO QUÍMICO (~5 questões)

### Constante de Equilíbrio (Kc)
Para aA + bB ⇌ cC + dD: **Kc = [C]^c[D]^d / [A]^a[B]^b**

| Kc | Significado |
|----|-------------|
| > 1 | Produtos favorecidos (→) |
| < 1 | Reagentes favorecidos (←) |

### Le Chatelier
> Sistema perturbado se desloca para minimizar a perturbação.

| Perturbação | Deslocamento |
|-------------|-------------|
| ↑ [reagente] | → direita |
| ↑ T (exotérmica ΔH<0) | ← esquerda |
| ↑ T (endotérmica ΔH>0) | → direita |
| ↑ P (com gás) | lado com MENOS mols |
| Catalisador | Não desloca! Só acelera |

### pH e pOH
- pH = -log[H⁺] | pOH = -log[OH⁻]
- pH + pOH = 14
- [H⁺][OH⁻] = 10⁻¹⁴
- pH < 7 = ácido. pH = 7 = neutro. pH > 7 = básico.

**Cálculo rápido:** [H⁺]=10⁻³ → pH=3. [H⁺]=5×10⁻³ → pH=3-log5=3-0,7=2,3

### Hidrólise
- **Ác. forte + Base fraca** → pH ácido (ex: NH₄Cl)
- **Ác. fraco + Base forte** → pH básico (ex: NaHCO₃)
- **Tampão:** ác. fraco + sal (pH = pKa + log([sal]/[ácido]))

## 2. CINÉTICA QUÍMICA (~2 questões)

### Fatores que afetam velocidade
- ↑ [ ] → ↑ v
- ↑ T → ↑↑ v (≈ dobra a cada 10°C)
- ↑ superfície → ↑ v
- Catalisador → ↑ v (↓ energia de ativação)

### Lei da Velocidade: v = k[A]^m[B]^n
- Ordem 0: v = k (independente de [A])
- Ordem 1: v = k[A] (v dobra se [A] dobra)
- Ordem 2: v = k[A]² (v quadruplica se [A] dobra)

## 3. TERMOQUÍMICA (~2 questões)

- **Exotérmica:** ΔH < 0 (libera calor)
- **Endotérmica:** ΔH > 0 (absorve calor)

### Lei de Hess
> ΔH depende só dos estados inicial e final. Some as equações.

### Entalpia de Ligação
ΔH = Σ(ligações quebradas) - Σ(ligações formadas)

## 4. ELETROQUÍMICA (~1 questão)

### Pilha
| | Ocorre | Polo |
|---|--------|------|
| **Ânodo** | **Oxidação** | Negativo (-) |
| **Cátodo** | **Redução** | Positivo (+) |

**Macete:** Ânodo = Oxidação (ambos começam com vogal).
**DDP:** ΔE = E°cátodo - E°ânodo. ΔE > 0 = espontânea.

### Eletrólise
- Processo NÃO espontâneo (usa energia elétrica)
- Lei de Faraday: m = (M×i×t)/(n×F), F=96500 C/mol
